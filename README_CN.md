## ✨ 功能亮点

### 🎤 实时变声器
- **7种变声效果**：原声、萝莉、大叔、机器人、电音、花栗鼠、深沉
- **实时处理**：边说边变声，无需等待
- **简单易用**：一键选择，按住说话

### 🔊 音效板
- **8种常用音效**：鼓掌、笑声、哇哦、欢呼、悲伤、生气、加油、惊讶
- **即时播放**：点击即播，无需等待
- **震动反馈**：增强操作手感

### ⚡ 悬浮工具箱
- **可拖拽悬浮球**：自由移动位置，不遮挡游戏画面
- **6种快捷工具**：亮度、触感、勿扰、网络、录屏、截图
- **边缘吸附**：自动贴合屏幕边缘

## 🚀 快速开始

### 第一步：获取 IPA 文件

1. **Fork 本仓库**
2. 进入 **Actions** 页面
3. 点击 **"Build iOS IPA" → "Run workflow"**
4. 等待 3-5 分钟
5. 下载生成的 IPA 文件

详细教程请查看 [快速开始指南](QUICKSTART.md)

### 第二步：安装到 iOS 设备

1. 将 IPA 文件发送到已安装 TrollStore 的 iOS 设备
2. 打开 TrollStore
3. 点击 "+" 按钮
4. 选择 IPA 文件
5. 等待安装完成

### 第三步：开始使用

1. 打开应用，授予权限
2. 选择功能：实时变声 / 音效板 / 悬浮工具箱
3. 享受游戏乐趣！
