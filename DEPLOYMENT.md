# 秋葵游戏助手 - 使用部署指南

## 📦 如何获取 IPA 文件

### 方法一：通过 GitHub Actions 自动构建（推荐）

这是最简单的方法，无需任何 macOS 设备。

#### 步骤：

1. **Fork 本仓库**
   - 访问本项目页面
   - 点击右上角的 "Fork" 按钮
   - 将仓库 fork 到你的 GitHub 账号

2. **手动触发构建**
   - 进入你 fork 的仓库
   - 点击 "Actions" 标签
   - 找到 "Build iOS IPA" workflow
   - 点击 "Run workflow" → 选择分支 → 点击绿色 "Run workflow" 按钮

3. **等待构建完成**
   - 构建过程大约需要 3-5 分钟
   - 你可以在 Actions 页面实时查看构建进度

4. **下载 IPA 文件**
   - 构建完成后，点击该次运行记录
   - 向下滚动找到 "Artifacts" 区域
   - 下载 `AutumnOkraGameAssistant-IPA` 文件
   - 解压后即可得到 `.ipa` 文件

### 方法二：本地编译（需要 Mac）

如果你有 macOS 设备，也可以自己编译：

#### 步骤：

1. **安装 Xcode**
   - 从 App Store 下载并安装最新版 Xcode（15.0+）
   - 安装完成后打开 Xcode 同意许可证

2. **克隆或下载项目**
   - 使用 git clone 或直接下载 ZIP 文件

3. **打开项目**
   - 双击 `AutumnOkraGameAssistant.xcodeproj` 打开项目

4. **选择目标设备**
   - 点击顶部工具栏的目标设备选择器
   - 选择 "Any iOS Device"（用于生成 IPA）

5. **导出 IPA**
   - 在菜单栏选择 `Product` → `Archive`
   - 等待编译完成
   - 在 Organizer 窗口中选择生成的 Archive
   - 点击 "Distribute App"
   - 选择 "Ad Hoc" 或 "Development"
   - 导出 IPA 文件

## 🚀 如何安装 IPA 文件

### 使用 TrollStore 安装

TrollStore 是最推荐的安装方式，因为它不需要开发者证书，可以永久使用。

#### 前提条件：
- 已安装 TrollStore 的 iOS 设备
- TrollStore 版本需要支持 IPA 安装

#### 安装步骤：

1. **将 IPA 文件传输到 iOS 设备**
   - 方式一：通过 AirDrop 从 Mac 传输
   - 方式二：通过微信/QQ 等应用发送
   - 方式三：通过 iCloud Drive 同步
   - 方式四：使用 Filza 等文件管理器

2. **使用 TrollStore 安装**
   - 打开 TrollStore 应用
   - 点击 "+" 按钮
   - 找到并选择下载的 `.ipa` 文件
   - 等待安装完成
   - 应用会出现在主屏幕

3. **首次运行**
   - 点击应用图标启动
   - 授予麦克风和相册权限（按提示操作）
   - 开始使用！

## 🎮 使用教程

### 实时变声器使用：

1. **选择变声效果**
   - 打开应用，点击 "实时变声"
   - 在效果列表中选择你想要的声音（萝莉、大叔、机器人等）

2. **录制并变声**
   - 按住底部的红色按钮开始录音
   - 说话的同时实时应用变声效果
   - 松开按钮停止录音并自动播放

3. **调整效果**
   - 可以尝试不同的变声效果
   - 萝莉效果适合俏皮语音
   - 深沉效果适合严肃场景

### 音效板使用：

1. **播放音效**
   - 点击 "音效板"
   - 直接点击任意音效按钮即可播放
   - 支持连续点击播放多个音效

2. **常用场景**
   - 鼓掌：队友表现好时
   - 笑声：发生有趣的事情
   - 哇哦：惊叹时刻
   - 欢呼：胜利时刻

### 悬浮工具箱使用：

1. **开启悬浮球**
   - 点击 "悬浮工具箱" 按钮
   - 悬浮球会出现在屏幕右侧

2. **移动悬浮球**
   - 长按悬浮球可以拖拽
   - 松手后会自动吸附到屏幕边缘

3. **使用工具**
   - 点击悬浮球打开工具菜单
   - 点击对应功能即可使用：
     - 亮度：快速调整屏幕亮度
     - 触感：触发震动反馈
     - 勿扰：跳转到系统勿扰设置
     - 网络：查看当前网络状态
     - 录屏：快速启动录屏
     - 截图：一键保存截图到相册

## ⚠️ 常见问题

### 1. GitHub Actions 构建失败

**原因**：GitHub Actions 有使用限制

**解决**：
- 检查仓库是否为公开仓库（公开仓库有更多配额）
- 等待几分钟后重试
- 或使用本地编译方式

### 2. TrollStore 安装失败

**原因**：TrollStore 版本过旧或 IPA 文件有问题

**解决**：
- 更新 TrollStore 到最新版本
- 重新下载或编译 IPA 文件
- 检查 iOS 版本是否兼容（需要 iOS 13.0+）

### 3. 应用闪退或无法启动

**原因**：权限未授予或系统不兼容

**解决**：
- 检查是否授予了麦克风权限
- 确认 iOS 版本在 13.0 以上
- 尝试重新安装应用

### 4. 变声效果不明显

**原因**：设备性能或音频设置问题

**解决**：
- 在设置中检查系统音频设置
- 尝试不同的变声效果
- 确保麦克风没有被其他应用占用

### 5. 悬浮球不显示

**原因**：系统限制或应用未在后台运行

**解决**：
- 确保应用没有被系统杀死
- 在设置中允许应用在后台运行
- 重新打开应用并点击悬浮工具箱
