# 秋葵游戏助手 - 快速开始

## 🚀 5 分钟快速上手

### 第一步：获取 IPA 文件

**最简单的方法（无需 Mac）：**
1. Fork 本仓库到你的 GitHub
2. 进入仓库的 "Actions" 页面
3. 点击 "Build iOS IPA" → "Run workflow"
4. 等待 3-5 分钟，下载生成的 IPA 文件

### 第二步：安装到 iPhone/iPad

1. 将下载的 IPA 文件发送到你的 iOS 设备
   - 可通过：AirDrop / 微信 / QQ / iCloud / 文件管理器

2. 打开已安装的 **TrollStore**
3. 点击右上角 "+" 按钮
4. 选择刚传输的 IPA 文件
5. 等待安装完成

### 第三步：启动应用

1. 在主屏幕找到 "秋葵游戏助手" 图标
2. 点击打开
3. 授予麦克风和相册权限（按提示操作）

### 第四步：开始使用

**🎤 实时变声：**
- 点击"实时变声"
- 选择"萝莉"、"大叔"、"机器人"等效果
- 按住按钮说话，松开自动播放变声效果

**🔊 音效板：**
- 点击"音效板"
- 点击任意按钮播放音效（鼓掌、笑声、欢呼等）
- 支持连续点击

**⚡ 悬浮工具箱：**
- 点击"悬浮工具箱"
- 屏幕出现悬浮球
- 点击悬浮球打开工具菜单（亮度、触感、勿扰、网络、录屏、截图）

## 📱 常见问题快速解决

### ❌ 应用无法安装？
**解决**：
- 确保已安装 TrollStore
- 更新 TrollStore 到最新版本
- 重新下载 IPA 文件

### ❌ 变声效果听不到？
**解决**：
- 检查是否授予了麦克风权限
- 在系统设置中检查音量
- 尝试其他变声效果

### ❌ 悬浮球不显示？
**解决**：
- 确保应用在前台运行
- 点击"悬浮工具箱"按钮重新开启
- 检查是否被系统限制了

### ❌ 构建失败？
**解决**：
- 如果使用 GitHub Actions，等待几分钟后重试
- 或使用本地编译方式（需要 Mac）
